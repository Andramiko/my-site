About me

Halo, saya Andramiko, dan saya senang bisa berbagi sedikit tentang diri saya dengan Anda di sini. Saya adalah seorang Mahasiswa sekaligus personal trainer, dan saya telah menjadi personal trainer selama 3 tahun. Saya percaya bahwa pekerjaan saya adalah panggilan saya, dan saya sangat menikmati setiap momen dari pekerjaan saya.

Sebagai seorang profesional, saya sangat fokus pada kualitas kerja saya. Saya selalu mencoba untuk memberikan hasil terbaik yang saya bisa, dan saya senang melihat klien saya senang dengan hasil kerja saya. Saya juga sangat bersemangat tentang mencari cara baru untuk meningkatkan keterampilan saya dan memperluas pengetahuan saya di bidang ini. Sekian sedikit tentang saya. Saya berharap dapat memiliki kesempatan untuk bertemu dan bekerja dengan Anda di masa depan. Terima kasih telah mengunjungi portofolio saya!.
